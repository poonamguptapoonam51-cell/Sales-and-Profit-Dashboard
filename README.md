# 📊 Sales & Profit Dashboard (Excel)

## 🚀 Project Story
Initially, this dataset looked like just rows and columns with no clear meaning.  
But the goal was to understand what the data is actually trying to say.

I started by cleaning and organizing the raw data, then used Excel tools to analyze it step by step.  
As the analysis progressed, the data started revealing real business insights.

## 🔍 Key Insights
- Total Sales crossed ₹30L  
- Total Profit reached ₹26L+  
- Food category performed the best  
- South region led overall sales  
- Furniture category showed low performance  

## 🛠 Tools & Techniques Used
- Microsoft Excel  
- Data Cleaning  
- Pivot Tables  
- Pivot Charts  
- Slicers (for interactivity)  

## 📈 Dashboard Features
- Category-wise analysis  
- Region-wise performance  
- Sales & Profit overview  
- Interactive filters using slicers  

## 💡 Key Learning
This project helped me understand that:
> Raw data is not useful until it is converted into meaningful insights.
